#  Sub-Zero Wiki
### *The Encyclopedia for the Rest of Us.*

Sub-Zero Wiki is a prestige-focused platform dedicated to documenting the lives of people who are absolutely not famous. If you have a Nobel Prize, you're banned. If you have a blue checkmark, don't even look at us.

##  How it Works
This app uses the GitHub API to "verify" your lack of notability. 
- **Followers < 100:** You get a full Wikipedia-style entry.
- **Followers > 500:** You are "Too Famous" and rejected by our editors.

##  Tech Stack
- **Backend:** Python / Flask
- **Frontend:** Tailwind CSS (Wiki-UI Theme)
- **Data Source:** GitHub Public API
